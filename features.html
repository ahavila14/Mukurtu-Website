Mukurtu Features

  • Professional templates (color schemes, layout, navigation)
  • Adaptable cultural protocols (set your own community's protocols to match user access levels)
  • Extensible metadata fields support traditional knowledge concepts
  • Easy-to-use administration pages for customization and management
  • Step-by-step user experience to add digital heritage
  • Flexible seach fields for filtering content
  • Intuitive licensing tags/documentation for all content (at item and collections levels) tailored to the needs of Indigenous cultures
  • Innovative licensing allows you to license materials using new traditional knowledge licenses, Creative Commons or traditional copyright
  • Web-publishing and syndication options (RSS, ATOM)

Upcoming Features

  • Custom keyboards for data input in Indigenous languages
  • Integrated mapping--link content to Google or Bing maps
  • XML-based export/import system for interaction with collecting institutions (including crosswalks to MARC, DUBLIN CORE and other international standards)
  • On-the-fly audio recording for comments, translations and traditional knowledge

Now in Development

Mukurtu is partnering with Smallbean to create a robust mobile application which combines the audio, video, and camera capabilities of an Android smartphone into a simple yet robust application for use by Smallbean Citizen Archivists and other interested individuals seeking to record and archive oral history interviews and related cultural footage and newsworthy events. This app will allow:

  • Real-time, on-the-fly upload from remote locations to centralized Mukurtu archive
  • Bundling and export of multiple types of content file for ingestion into Mukurtu system
  • Upload metadata tracked across content types
  • Tagging features for all content
  • Cross-platform interoperability
  • Playback and view functionality
